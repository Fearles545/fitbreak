# FitBreak Logo Prompt

## Concept: "F Stretch in Monitor"

The letter F stylized as a person stretching — the vertical bar is the body, the two horizontal bars are arms reaching outward, slightly angled up like a side stretch. The stretching F sits inside a rounded rectangle representing a monitor/screen frame. The whole mark tells the story: exercise happening at your workspace.

## Prompt

Minimalist logo mark for "FitBreak", a health app that reminds desk workers to take exercise breaks. The letter F is stylized as a human figure stretching — the vertical stroke is the body, the two horizontal strokes are arms reaching outward and slightly angled upward like a side stretch. This stretching F is placed inside a rounded rectangle that represents a computer monitor or screen frame. Deep purple color (#5E35B1) on white background. Flat geometric style, no gradients, no shadows. Simple enough to read clearly at 16x16 pixels. Modern, friendly, professional. SVG-ready clean vector shapes.

## Required Exports

- SVG — scalable source, favicon
- PNG 512x512 — PWA manifest splash screen
- PNG 192x192 — PWA manifest install icon
- PNG 180x180 — Apple touch icon
- ICO 16x16 + 32x32 — legacy favicon fallback

## Recommended Tools

- **Recraft.ai** — best for clean vector/icon logos, exports SVG directly
- **Midjourney / DALL-E** — good for exploring concepts, vectorize after
- **Figma** — recreate manually for pixel-perfect control
